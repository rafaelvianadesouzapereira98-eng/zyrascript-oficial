Manual Oficial de Uso da HygeaScript (v4.0)

Referência completa da linguagem de programação Zyra.

1. Introdução

A HygeaScript é uma linguagem de alto nível projetada para ser simples e poderosa, com suporte para Playground (Web) e Terminal (CLI).

Download: Todos os arquivos (zyra_playground_v4.html, zyra.exe) estão disponíveis no site oficial.

2. Funções Essenciais (Comandos Nativos)

Comandos básicos que podem ser usados em qualquer parte do seu código Zyra.

ComandoDescriçãoExemplo
digaImprime texto ou o valor de variáveis na saída.diga "Olá, " Nome
soma, subtraiRealiza operações básicas.soma 10 5

3. Estruturas de Fluxo de Controle

3.1. Condicionais

Usada para tomar decisões no código.

se [CONDIÇÃO] então [BLOCO DE CÓDIGO] fim 
se Idade > 18 então
    diga "Maior de idade"
fim

3.2. Repetições (Loops)

Repete um bloco de código um número fixo de vezes.

repete [NÚMERO] vezes [BLOCO DE CÓDIGO] fim 
repete 3 vezes
    diga "Repetindo..."
fim

4. Variáveis, Funções e Listas

4.1. Variáveis e Atribuição

Use o sinal de = para armazenar qualquer valor.

Nome = "Zyra"
Resultado = 10 * 5

4.2. Funções Personalizadas

Permite criar blocos de código reutilizáveis.

func NomeFuncao(param1, param2) [BLOCO] retorna [VALOR] fim 
func MultiplicaPorDois(valor)
    NovoValor = valor * 2
    retorna NovoValor
fim

X = MultiplicaPorDois(50)
diga X # Imprime 100

4.3. Listas (Arrays)

Cria e acessa coleções de dados.

ListaFrutas = lista ["Maçã", "Banana", 176]
diga ListaFrutas[0] # Imprime "Maçã"
